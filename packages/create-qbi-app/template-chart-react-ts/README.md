# Quick BI Custom Chart

## Setup

Install the dependencies:

```bash
npm install # or pnpm install or yarn install
```

## Get started

Start dev server:

```bash
npm run start
```

Build the component:

```bash
npm run build
```

Bundle the component:

```bash
npm run bundle
```
